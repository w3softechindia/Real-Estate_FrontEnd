Overview Official Website

Swiper is the most modern slider with hardware accelerated transitions and amazing native behavior.

Usage

Swiper Slider's CSS and Javascript files are bundled in the vender.min.css and vendor.js and globally included in all pages.

Default Swiper #

set swiperConfig to set a default swiper.

Navigation & Pagination Swiper #

Use modules="navigation" attribute to set a swiper config with navigation and pagination.

Pagination Dynamic Swiper #

Use modules="Pagination" attribute to set a swiper config with dynamic and pagination.

Effect Fade Swiper #

Use effect="fade" attribute to set a swiper config with fade effect.

Effect Creative Swiper #

Use effect="creative" attribute to set a swiper config with creative custom effect.

Effect Flip Swiper #

Use effect: 'flip' attribute to set a swiper config with flip custom effect.

Scrollbar Swiper #

Use scrollbar: el: '.swiper-scrollbar',hide: true, attribute to set a swiper config with scrollbar pagination.

Vertical Swiper #

Use direction: 'vertical' attribute to set a swiper config with vertical pagination.

Mousewheel Control Swiper #

Use mousewheel: true, attribute to set a swiper config with mousewheel scroll.

Responsive Breakpoints Swiper #

set breakpoints attribute to swiper config set a responsive swiper.
